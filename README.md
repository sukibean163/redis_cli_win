# redis_cli_win
## redis windows client example<br>
    with hiredis build 
    [reference: https://github.com/lgsonic/hiredis-win](https://github.com/lgsonic/hiredis-win)
### build tool
    cmake
### how to use:<br>
```
mkdir build
cd build
cmake .. -G"Visual Studio 10 2010" or cmake .. -G"Visual Studio 11 2012"
or 
cmake .. -G"MinGW Makefiles"  
```
